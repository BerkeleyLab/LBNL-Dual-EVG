Bootstrapping A New Board

These instructions present the procedure to download and execute FPGA firmware and software using its JTAG port.  The instructions assume that a download image is ready for download as described in the Updating Firmware documentation and that a Vivado session with the desired project is running.
  1. Connect the USB ports of the development machine and the FPGA to be programmed.
  2. In a terminal window start the Xilinx virtual cable server:
        ftdiJTAG -c 30M -g 11
  3. In the Vivado session click 'Open Hardware Manager' (or Flow→Hardware Manager menu item).
  4. Click 'Open Target'.
  5. Click 'Open New Target...'.
  6. In the 'Open New Hardware Target' window that pops up, click 'Next>'.
  7. Set the menu to 'Connect to Local server', then click 'Next>'.  This menu item must be chosen even if the server was started on a different machine.
  8. If the desired device is not present in the 'Hardware Targets' table, click the 'Add Xilinx Virtual Cable (XVC)' button and select the IP address and port corresponding to the server started in step 2 (defaults are 127.0.0.1 and 2542, respectively).
  9. Ensure that the desired device is highlighted in the 'Hardware Targets' table, then click 'Next>'.
  10. Click 'Finish'.
  11. Click 'Program Device' (or Tools→Program Device menu item).
  12. In the 'Program Device' window that pops up, clear the 'Debug probes file' field then navigate the 'Bitstream file' field to the scripts directory in the Vitis workspace, select the download.bit file there, then click 'Program'.
  13. Once the download is complete the application will start running.  Use a console session in another terminal window to observe its progress.
  14. Use a TFTP client to transfer the download.bit file to the flash memory DEVG_A.bit.  There is a 'programFlash.sh' script in the scripts directory to assist with this.
  15. Set the application system parameters in flash memory as appropriate.
Steps 1 through 10 apply to initiating a ChipScope session, too.  Perform them, then click 'Refresh Device'.
